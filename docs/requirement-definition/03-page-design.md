### 機能説明

**ナビゲーション:**
- 年間分析: 年次データの傾向とサマリー表示
- 月間分析: 月次詳細データとグラフ表示  
- 投稿分析: 個別投稿のパフォーマンス分析

**アカウント切り替え:**
- アイコン + ユーザー名をクリックでポップオーバー表示
- 複数のInstagramアカウントを管理・切り替え可能
- 新規アカウント追加機能

### 投稿分析ページ機能説明

**フィルター機能:**
- 期間指定（カレンダー選択）
- 投稿タイプ別フィルタリング（All, IMAGE, VIDEO, CAROUSEL, STORY）

**投稿一覧テーブル:**
- 各行が1つの投稿を表示
- **投稿日**: 投稿された日付
- **サムネイル**: 投稿の画像プレビュー
- **タイプ**: カルーセル、動画、画像、ストーリーの種別
- **リーチ**: 投稿を見たユニークユーザー数
- **いいね**: いいね数
- **コメント**: コメント数  
- **シェア**: シェア数
- **保存**: 保存数
- **EG率**: エンゲージメント率（パーセンテージ）
- **視聴数**: 動画の場合の視聴数
- **視聴率**: 動画の場合の視聴率

**パフォーマンス推移グラフ:**
- 時系列でのエンゲージメント率推移
- いいね数、リーチ数、視聴数の棒グラフ
- エンゲージメント率の折れ線グラフ
- 複数指標の重ね合わせ表示

### 年間分析ページ機能説明

**期間表示:**
- ページ上部に分析対象期間を表示（2022/01/01 - 2025/08/31）

- **ウェブサイトクリック**: 月間ウェブサイトクリック数
- **いいね**: 月間総いいね数
- **コメント**: 月間総コメント数
- **シェア**: 月間総シェア数

**フォロワー推移グラフ:**
- 月別フォロワー数とフォロー数の折れ線グラフ
- 時系列での成長推移を可視化

**エンゲージメント分析グラフ:**
- いいね、コメント、シェア、保存の積み上げ棒グラフ
- 月別エンゲージメントの内訳を表示

### セットアップページ機能説明

**Instagram API 設定フォーム:**
- **App ID**: Instagram App ID を入力
- **App Secret**: Instagram App Secret を入力  
- **Access Token**: Instagram Access Token を入力
- **トークン更新ボタン**: 入力されたCredentialsでトークンを更新・リフレッシュ

**アカウント連携状況表示:**
- 接続済みの Instagram アカウント一覧を表示
- 各アカウントの詳細情報:
  - **アカウント名**: @username形式で表示
  - **トークン状況**: 有効/期限切れ/未設定
  - **有効期限**: トークンの有効期限日時
  - **最終更新**: 最後にトークンを更新した日時
  - **ステータス**: 正常/更新が必要/未連携のアイコン表示

**機能仕様:**
- 新しいCredentialsを入力してトークン更新を実行
- トークンリフレッシュ後、アカウント連携状況が自動更新
- 期限切れアカウントの警告表示
- 未連携アカウントの識別表示
