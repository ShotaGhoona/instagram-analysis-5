### 毎日実行APIコール
```
GET /{ig-user-id}?fields=id,username,name,profile_picture_url,followers_count,follows_count,media_count
```

### データベース保存項目
- `date`: 収集日
- `ig_user_id`: Instagram User ID  
- `username`: ユーザー名
- `name`: 表示名
- `profile_picture_url`: プロフィール画像URL
- `followers_count`: フォロワー数
- `follows_count`: フォロー数
- `media_count`: 投稿数
